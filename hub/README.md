## ⚡ CORE DIRECTIVES

The **Macroflow Hub** is the central nervous system for the Hyatt Ecosystem. Built on a cutting-edge React/Vite stack with a cinematic dark-mode/Matrix aesthetic, it serves as the ultimate command center orchestrating multiple autonomous systems, dynamic repositories, and embedded applications without ever forcing the user to leave the environment.

### 🌌 KINETIC INTERFACE
- **Fanned Deck Architecture:** A 2D rotating fanned stack built with Framer Motion. Uses advanced momentum-based swiping and scroll-wheel physics to instantly traverse dozens of dynamically loaded GitHub repositories.
- **Embedded Matrix Modals:** Clicks intercept external navigation, sliding up a stylized Matrix window that fetches live `README.md` markdown directly from the GitHub API and renders it via `react-markdown` and Tailwind Typography.
- **Dynamic Routing:** Internal React Router configurations instantly mount iframes of external or local applications, keeping the entire ecosystem fully encapsulated.
- **Galaga Background Engine:** Custom HTML5 canvas masking with `screen` blend modes injecting seamless, animated retro aesthetics.

### 🛠 TECH STACK
- **Framework:** React 19 + Vite
- **Routing:** React Router v7
- **Styling:** TailwindCSS v4
- **Animation:** Framer Motion (Spring-loaded physics, drag handling)
- **Parsing:** React-Markdown + Remark-GFM
